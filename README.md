# Patient Dashboard

A React-based healthcare dashboard for managing patient information, vitals, diagnostics, and lab results.

## Features

- Patient list with profile information
- Diagnosis history with blood pressure charts
- Vital signs monitoring (Respiratory Rate, Temperature, Heart Rate)
- Diagnostic list with problem/diagnosis tracking
- Lab results management

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file in the root directory:
```
REACT_APP_API_URL=https://fedskillstest.coalitiontechnologies.workers.dev
REACT_APP_USERNAME=your_username
REACT_APP_PASSWORD=your_password
```

3. Start the development server:
```bash
npm start
```

## Build

To build for production:
```bash
npm run build
```
